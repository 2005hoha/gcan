### Q2 核心：GCAN 数据流

```
  obs [B,T,N,obs_dim]
        │
        ▼
  ┌─────────────┐     ┌──────────────────┐     ┌─────────────────┐
  │ GraphBuilder │────▶│ 2-layer GATConv  │────▶│  CreditHead     │
  │  (全连接图)   │     │  (4 heads→1head) │     │  Softmax → [N,1]│
  └─────────────┘     └──────────────────┘     └────────┬────────┘
                                                        │
  agent_qs [B,T,N] ────────────────────────────────────┤
                                                        ▼
                                              Q_tot = Σ(credit_i × q_i) + V(state)
                                              
  对比 QMIX:
  state → HyperNetwork → abs(weights) × agent_qs → Q_tot (单调约束)
  GCAN:  graph → GAT attention → softmax(credits) × agent_qs → Q_tot (无单调约束)
```
